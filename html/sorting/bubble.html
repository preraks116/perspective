Bubble Sort

Simulation

The bubble sort algorithm is a simple sorting algorithm that works the way we sort playing cards in our hands. You imagine a row of n cards where the ith card is showing the number i. You take the top card and place it in the correct position by swapping it with the card immediately above it. You then take the second card and place it in the correct position by swapping it with the card immediately above it. You continue this process until you have sorted the entire row.

Explanation

Bubble sort, sometimes referred to as sinking sort, is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted. The algorithm, which is a comparison sort, is named for the way smaller or larger elements "bubble" to the top of the list.

Bubble sort is a comparison sort, meaning that it can sort items of any type for which a total order is defined.

The algorithm consists of two phases: the sorting phase and the checking phase. The sorting phase moves through the list looking for adjacent pairs of items that are out of order, and swaps them. The checking phase compares each pair of items to make sure they are in the correct order.

The algorithm is a simple and intuitive to understand. It works by repeatedly stepping through the list, comparing adjacent items and swapping them if they are in the wrong order.

The algorithm is not efficient for large lists. It is generally only efficient for lists where the number of items is much smaller than the number of passes through the list.

The algorithm is not stable; two items may be swapped when they are compared.

The algorithm is not adaptive; it does not consider whether the list is already sorted.

The algorithm is not in-place; it requires extra space for the temporary storage of the swapped items.

Code


    # Creating a bubble sort function  
    def bubble_sort(list1):  
        # Outer loop for traverse the entire list  
        for i in range(0,len(list1)-1):  
            for j in range(len(list1)-1):  
                if(list1[j]>list1[j+1]):  
                    temp = list1[j]  
                    list1[j] = list1[j+1]  
                    list1[j+1] = temp  
        return list1  

    list1 = [5, 3, 8, 6, 7, 2]  
    print("The unsorted list is: ", list1)  
    # Calling the bubble sort function  
    print("The sorted list is: ", bubble_sort(list1))
