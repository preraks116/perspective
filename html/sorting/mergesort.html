Merge Sort

Simulation

The blue bars that you see above represent the unsorted list of numbers, where if one bar on the left is taller than another on the right, it means that the number on the left is larger than the number on the right.

The green bars that you see below represent the sorted list of numbers. Since merge sort is a recursive algorithm, it will split the list into smaller lists until it reaches the base case, where it will then merge the smaller lists back together. You can see each of those small lists as sections of green bars which are being merged into the larger list.

Explanation

Merge sort is a Divide and Conquer algorithm that splits the list into smaller lists until it reaches the base case, where it will then merge the smaller lists back together. The algorithm is recursive, so it will split the list into smaller lists until it reaches the base case, where it will then merge the smaller lists back together.

We define the base case is when the list is only one element long.

The steps of the algorithm are:

  1. While the list is not one element long, split the list into two smaller lists.
  2. Split the list into two smaller lists. The split is done in the middle of the list.
  3. Then we will sort the smaller lists. This is where the function is called recursively.
  4. Merge the sorted lists. This involves comparing elements from both the lists one by one and placing the smaller element in the sorted list one by one.

Complexity

The time complexity of merge sort is O(n log n), where n is the number of elements in the list.

The space complexity of merge sort is O(n), where n is the number of elements in the list.

# MergeSort in Python

def mergeSort(array):
    if len(array) > 1:

        #  r is the point where the array is divided 
        # into two subarrays
        r = len(array)//2
        L = array[:r]
        M = array[r:]

        # Sort the two halves
        mergeSort(L)
        mergeSort(M)

        i = j = k = 0

        # Until we reach either end of either L or M, 
        # pick larger among

        # elements L and M and place them in the 
        # correct position at A[p..r]
        while i < len(L) and j < len(M):
            if L[i] < M[j]:
                array[k] = L[i]
                i += 1
            else:
                array[k] = M[j]
                j += 1
            k += 1

        # When we run out of elements in either L or M,
        # pick up the remaining elements and put in A[p..r]
        while i < len(L):
            array[k] = L[i]
            i += 1
            k += 1

        while j < len(M):
            array[k] = M[j]
            j += 1
            k += 1

def printList(array):
    for i in range(len(array)):
        print(array[i], end=" ")
    print()

mergeSort(array)
